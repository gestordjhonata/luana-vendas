# Luana Vendas

> Contexto do projeto e guia inicial para a nova página da cliente Luana.

## O que este repo contém

- `index.html`: página inicial de apresentação e CTA.
- `CLAUDE.md`: âncora de documentação e orientação para iniciar o projeto.
- `README.md`: visão geral rápida do repositório.

## Objetivo

Criar uma página simples e de alta conversão para a Luana, com foco em:

- mensagem clara ao visitante
- chamada para ação (WhatsApp, formulário ou botão de vendas)
- aparência moderna e leve
- fácil edição futura

## Como usar

1. Abra `index.html`.
2. Substitua o conteúdo de exemplo por informações reais da cliente.
3. Adicione imagens em uma pasta `assets/` se precisar.
4. Se já tiver uma plataforma de vendas, insira o link/call-to-action no botão.

## Próximos passos

- Personalizar o título, o subtítulo e o texto do botão.
- Adicionar `assets/` com logo, fotos ou ilustrações.
- Criar uma segunda página de vendas ou formulário, se necessário.
- Se quiser, posso criar a estrutura de `assets/`, `styles.css` e um layout de checkout.
